1. Navigation

The main functions are summarised below:

next >>

Takes you to the next page

<< previous

Takes you to the previous page

Top of page

Takes you to the top of the current page

References

Opens up a separate window containing resources mentioned in the module

The main site menu on the left is collapseable, allowing you to view all sections and subsections of the Case Study. As well as the menu you can use the next >> and << previous buttons to navigate through the pages.

2. Requirements

In order to run the Case Study your pc will need to have any one of the following web browsers installed on your pc:

  • Internet Explorer
  • Firefox
  • Safari
  • Camino

You will need Flash 9 installed to view the animations and videos in the Case Study. You will also need Shockwave installed. Both of these can be downloaded from the Adobe website. A broadband internet connection is recommended in order to view these animations.
